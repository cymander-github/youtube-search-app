# YouTube Search Application

YouTube 동영상 검색 애플리케이션입니다.

## 기능
- YouTube 동영상 검색
- Shorts 영상 제외
- 조회수, 구독자수, 최신순 기반 정렬
- 캐시 시스템 적용

## 기술 스택
- Google Apps Script
- YouTube Data API v3
- HTML/CSS/JavaScript

## 설치 및 실행 방법
1. Google Cloud Console에서 YouTube Data API v3 활성화
2. Google Apps Script에서 프로젝트 생성
3. 코드 복사 및 배포
4. 웹 앱으로 실행
